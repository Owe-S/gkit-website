# Golfklubbens IT - Website & Marketing Ecosystem

Welcome to the official repository for the Golfklubbens IT website and marketing ecosystem. This project combines a modern React website with a "Google Native" backend and sales automation system.

## üöÄ Quick Start

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
npm install
```

### Development
Start the local development server:
```bash
npm run dev
```
Visit `http://localhost:5173` to see the site.

### Production Build
Create the production bundle for deployment:
```bash
npm run build
```
The output will be in the `dist` folder.

### Documentation
View and update the documentation:
```bash
# Preview docs locally
mkdocs serve

# Deploy docs to GitHub Pages (after code commit)
./scripts/deploy-docs.sh  # macOS/Linux
# or
scripts/deploy-docs.bat   # Windows
```

See `.github/DOCS-UPDATE-CHECKLIST.md` for documentation workflow.

## üåê Marketing Ecosystem

This project includes a complete marketing strategy built on Google tools.

### 1. The Website
- **Home Page**: Corporate landing page showcasing services and projects.
- **The 2026 Club**: Exclusive campaign landing page (`/2026`).

### 2. The "Invisible" CMS
- **Blog Content**: Managed via Google Sheets.
- **Script**: `cms_script.gs` (Deploy as Web App).

### 3. Sales Automation
- **CRM**: Managed via Google Sheets.
- **Automation**: `crm_automation.gs` handles form submissions and emails.
- **Analytics**: GA4 initialized in `App.tsx`.

### 4. Campaign Assets
- **Manifesto**: `the_2026_club_manifesto.md`
- **Video Script**: `video_script_2026.md`
- **Lead Magnet**: `lead_magnet_checklist.md`
